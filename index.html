Confección y Publicación de Páginas Web

EL OBJETIVO

Como competencia general, al acabar el curso podrás crear y publicar páginas web que integren textos, imágenes y otros elementos,utilizando lenguajes de marcas y editores apropiados, según especificaciones y condiciones de "usabilidad" dadas y realizar los procedimientos de instalación y verificación de las mismas en el servidor correspondiente

Gracias a nuestros profesores Anna Hernández Adeva,con una trayectoria profesional muy competitiva , y Rafael Mir Callejas,con su experiencia exepcional en diseño Grafico, aprenderas desde cero toda una profesión, la más pedida en la actualidad.

TEMÁTICA DEL CURSO

Creación de páginas web con el lenguaje de marcas y Elaboración de hojas de estilo

Con esta unidad informativa, serás capaz de realizar paginás web para presentar información como textos,imágenes,elementos de multimedia,entre otros utilizando herramientas de edición web como Visual Studio Code. HTML es un lenguaje de marcado que se utiliza para lograr este objetivo. Se trata de la siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto”.

Leer más...

Después de crear todos los elementos necesarios,toca darles estilo. Aprenderás como elaborar una hoja de estilo CSS;Las hojas de estilo son conjuntos de instrucciones se ocupan de los aspectos de formato y de presentación de los contenidos: tipo, fuente y tamaño de letras, alineación,entre otros. También, aprenderás como usar plantillas como ejemplo BOOTSTRAP .El objetivo final es tener un diseño responsive que es capaz de adaptarse a cualquier dispositivo donde se visualice.

Programación con lenguajes de guión en páginas web y Pruebas de funcionalidad y optimización de paginás web

Para crear páginas web dinámicas en que incorpora efectos como texto que aparece y desaparece, animaciones acciones que se activan al pulsar botones y ventanas con mensajes de aviso al usuario, aprenderás a programar con lenguajes especificos como Javascript.

Leer más...

Javascript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, hace algo más que mostrar información estática para que la veas. Aprenderás a través de ejercicios prácticos usando PHPpara tener una página web estable y dinámica que ofrezca una buena experiencia de usuario. Mientras que los lenguajes del lado del cliente como HTML, CSS o JavaScript son interpretados primero por el navegador web en el momento de abrir una página, el código PHP se ejecuta en el servidor web. Allí, los scripts de PHP generan el código HTML que se envía después al navegador. Este no recibe el código real (el script de PHP), sino el resultado de la ejecución del mismo.

Crear la imagen corporativa con Adobe y sitio web con Wordpress

En esta unidad, aprenderás como usar los programas ADOBE para crear un contenido profesional que llame la atención del expectador.Usarás como tecnología PHOTOSHOP principalmente para el retoque de fotografías y gráficos. Su nombre en español significa "taller de fotos". Es conocido mundialmente.Con el, editas cualquier imagen y también crearás GIFs y mucho más. También,conocerás el programa Indesign para elaborar y construir una imagen corporativa de un proyecto.Es una aplicación de diseño editorial.

Publicación de páginas web

Mediante este procedimiento se pone a disposición de cualquier usuario con acceso a Internet las páginas web al pasar a estar alojadas físicamente en un ordenador servidor con acceso permanente desde Internet.

Es el último paso, se enseña como subir los archivos desde el equipo local al servidor. Este paso se puede realizar mediante un programa que utiliza el protocolo FTP como ejemplo Filezilla. Esta transferencia de archivos se realiza en modo autentificado .
